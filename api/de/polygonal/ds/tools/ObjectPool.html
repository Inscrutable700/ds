class ObjectPool<T>

package de.polygonal.ds.tools

An object pool with an unbounded size that creates new objects on-the-fly and stores them for repeated use.

Example:

Constructor

new (factory:Void ‑> T, dispose:T ‑> Void, ?maxNumObjects:Int)

@maxNumObjects The maximum allowed number of pooled object. If omitted, there is no upper limit.

Variables

growthRate:Int

size:Int

Methods

free ():Void

Destroys this object by explicitly nullifying all objects for GC'ing used resources. Improves GC efficiency/performance (optional).

inline get ():T

Acquires the next object from this pool or creates a new object if all objects are in use. To minimize object creation, return objects back to the pool as soon as their life cycle ends by calling this.put(). If this.size equals this.capacity(), this.get() still allocates a new object but does not pool it. This effectively disables pooling.

preallocate (numObjects:Int):Void

inline put (obj:T):Void

Returns the object x to the pool. Objects are pushed onto a stack, so this.get() returns x if called immediately after this.put().
